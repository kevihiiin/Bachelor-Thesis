# DESeq2
DESeq2 is used to analyse the differential gene expression. The following contains the gene count files and scripts
to setup and run DESeq2.

## Content
### `count_files`
Each time point has a folder containing the count files for each sample.

### `generate_deseq_input.py`
Script to generate the input matrix count file for DESeq2.

### `output`
Input matrix for DESeq2 generated by `generate_deseq_input.py`.

### R_notebook
R-notebook containing the scripts used to generate the DESeq2 output and code for visualization.
